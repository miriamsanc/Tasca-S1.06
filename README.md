# Sprint-1 Tasca S1.06: PHP AVANÇAT
Exercicis de PHP practicant php avançat.

## Tecnologies 
- PHP
- HTML
- CSS
- XAMPP

## Instal·lació
1. Clonar el repositori: git clone [Link](https://github.com/miriamsanc/Tasca-S1.06.git)
2. Instal·la XAMPP en el teu ordinador.
3. Obre el panell de control de XAMPP i prem start a Apache.
4. Copia la carpeta del repositori dins del directori xampp/htdocs
5. Obre el teu navegador web i accedeix a: http://localhost/Tasca-S1.06/Level 1/index.html
6. Omple el formulari i al premer el botó s'enviará la informació introduida al arxiu submit.php
